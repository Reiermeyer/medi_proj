React Props

Props sind Argumente welche in React Components übergeben werden. Sie werden durch HTML-Attribute übergeben. Sie sind wie Funktionsargumente in JavaScript und Attribute in HTML. Um Props in Components zu übergeben, nutzt man dieselbe Schreibweise wie bei HTML Attributen:

Beispiel

Füge ein Attribut "name" dem Element "Mensch" zu: 
const beispiel = <Mensch name="Mustermann" />;

Das Component erhält das Argument als props-Objekt:

Beispiel

Nutze das Attribut "name" im Component: 
import React from 'react';
import ReactDOM from 'react-dom';

class Mensch extends React.Component {
  render() {
    return <h2> Ich heiße {this.props.name}! </h2>;
    }
  }

const beispiel = <Mensch name="Mustermann" />;
ReactDOM.render(beispiel, document.getElementById('wurzel'));
Browser 
Ich heiße Mustermann!

Daten weiterreichen

Props kannst du auch nutzen um Daten, als Parameter, von einem Component zum anderen zu überreichen.

Beispiel

Überreiche das "name" vom Component Deutschland zum Component Mensch: 
import React from 'react';
import ReactDOM from 'react-dom';

class Mensch extends React.Component {
  render() {
    return <h2>Ich heiße {this.props.name}!</h2>;
  }
}

class Deutschland extends React.Component {
  render() {
    return (
      <div>
      <h1>Ich komme aus Deutschland!</h1>
      <Mensch name="Mustermann" />
      </div>
    );
  }
}

ReactDOM.render(<Deutschland />, document.getElementById('wurzel'));
Browser 
                
Ich komme aus Deutschland!

                
Ich heiße Mustermann!

Wenn du eine Variable übergeben willst, dann setzt du sie einfach in geschwungene Klammern:

Beispiel

Erstelle eine Variable, z.B. mit dem Namen "person", und übergebe diese an das Component Mensch: 
import React from 'react';
import ReactDOM from 'react-dom';

class Mensch extends React.Component {
  render() {
    return <h2>Ich heiße {this.props.name!</h2>;
  }
}

class Deutschland extends React.Component {
  render() {
  const person = "Mustermann";
    return (
      <div>
      <h1>Ich komme aus Deutschland!</h1>
      <Mensch name={person} />
      </div>
    );
  }
}

ReactDOM.render(<Deutschland />, document.getElementById('wurzel'));

Oder wenn du ein Objekt übergeben willst:

Beispiel

Erstelle ein Objekt namens "person" und übergebe dieses an das Component Mensch: 
import React from 'react';
import ReactDOM from 'react-dom';

class Mensch extends React.Component {
  render() {
    return <h2>Ich heiße {this.props.name.vorname}!</h2>;
  }
}

class Deutschland extends React.Component {
  render() {
  const person = {vorname: "Max", nachname: "Mustermann"}
    return (
      <div>
      <h1>Ich komme aus Deutschland!</h1>
      <Mensch name={person} />
      </div>
    );
  }
}

ReactDOM.render(<Deutschland />, document.getElementById('wurzel'));
Browser 
                
Ich komme aus Deutschland!

                
Ich heiße Max!

Props im Konstruktor

Wenn dein Component eine Konstruktor-Funktion hat, dann sollten die props immer zum Konstruktor und auch zum React.Component mithilfe der super() Methode übergeben werden.

Beispiel

class Car extends React.Component {
 constructor(props) {
    super(props);
 }
 render() {
    return <h2>Ich bin ein Mensch!</h2>;
 }
}

ReactDOM.render(<Mensch vorname="Max"/>, document.getElementById('wurzel'));

Achtung! React Props sind nur lesbar, ein Fehler würde auftreten, wenn man versucht den Wert zu ändern.
